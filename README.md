# Javascript Mini Projects to build Foundation and understand core concepts.

# 1.Rock Paper Scissors

Simple Rock Paper Scissors game using HTML, CSS and **JavaScript**.

## Features
- Score tracking
- LocalStorage
- Image based moves

# 2.TO-DO List

  Simple TO DO list page, With LIght-Dark Theme Toggle button.

  ## Features
  -Input Tasks
  -Complete,Delete buttons

# 3.COUNTER

  Increment and Decrement on Button click

  ## Features
  -Increment,Decrement and Clear button
  
# 4.BASIC CALCULATOR

  -Simple calculator performs standard arithmetic operations with a clean UI and responsive layout.

  ## Features
  -Basic operations: +, -, ×, ÷, %
  -Prevents multiple operators in sequence
  -Clear display (C button)
  -Grid layout
  -Simple and minimal UI

  ## Future updates
  -Add backspace (DEL) functionality
  -Add keyboard support
  -Improve Responsiveness (on multiple screens)
